문자열 입력할 때 """ """으로 쓰면 따옴표 중간에 어떤 형태로 쓰던 다 읽어와서 출력해준다.
문자열 변수 뒤에 [숫자]를 쓰는것으로 문자열에 인덱싱이 가능하다.
문자열 변수[이상:미만:간격] ->슬라이싱 
문자열 중간에 "%변수에 맞는 문자"를 넣으면 문자열의 따옴표 뒤에 %(문자열 사이에 들어갈 변수)를 써서 문자열 사이에 변수를 넣을 수 있다
비슷하게 문자열 사이에 {}를 넣고 따옴표 뒤에 .format()하고 괄호안에 중괄호에 넣고 싶은걸 적으면 된다

공백과 정렬
	문자열 입력할 때 %뒤에 숫자를 원하는만큼 적으면 적은대로 간격을 만들어 준다.
count : 문자열에 인자로 준 문자가 몇개 있는가 세어주는 함수
find : 문자열에 인자로 준 문자가 있는가 봐주는 함수 있으면 인덱스가 나오고 없으면 -1
join : 문자열에 인자를 삽입
upper : 대문자로 바꿔줌 
lower : 소문자로 바꿔줌
split : 문자열을 문자들로 나눠준다.
del : 리스트의 원소를 지워준다
append : 리스트의 마지막에 더해준다
sort : 리스트를 정렬해준다
reverse : 리스트를 뒤집어준다
index : 리스트에 ~가 있냐
insert : 리스트 몇번에 ~를 추가해줘
remove : 앞에서 부터 해당하는 1개를 제거해준다
pop : 마지막거가 추출된다
extend : 리스트에 리스트를 추가한다.
